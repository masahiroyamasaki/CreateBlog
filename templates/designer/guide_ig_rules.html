Instagram 投稿ルール

Instagram

コンテンツポリシー（禁止事項）

Instagramは Meta の「コミュニティガイドライン」に基づいて投稿を審査しています。以下を含む投稿は削除・アカウント停止の対象になります。

絶対に投稿してはいけないもの

  • 裸・性的コンテンツ（フィルタリングあり）
  • 暴力・残酷な描写
  • ヘイトスピーチ（人種・宗教・性別等による差別的発言）
  • 危険物の売買・製造方法
  • 自傷・自殺の促進コンテンツ
  • スパム・詐欺的コンテンツ（偽の情報、フォロワー購入宣伝など）
  • 著作権侵害（他者の写真・音楽・動画の無断使用）

投稿頻度の目安

投稿タイプ推奨頻度補足
フィード投稿週3〜5回毎日投稿でも問題ないが、質を優先
ストーリーズ毎日1〜3件24時間で消えるため頻度高め推奨
リール週2〜3回現在最もリーチが広がりやすい形式
Threads毎日1〜3投稿テキストなので作成コストが低い
このアプリのAPI投稿について
現在このアプリはフィード投稿（画像）のみ対応しています。ストーリーズ・リールは別途手動で行ってください。

垢BAN（アカウント停止）になるケース

アカウントが停止されると、それまでの投稿・フォロワーがすべて失われます。API経由の自動投稿でも同じルールが適用されます。

主な原因

NG — 短時間での大量アクション
1時間に大量のフォロー・いいね・コメントを行うとスパム判定されます。APIの自動投稿も1日数件程度に抑えるのが安全です。
NG — フォロワー・いいねの購入
フォロワーやいいねを購入するサービスの利用は規約違反です。発覚するとアカウントが永久停止される場合があります。
NG — 同一または類似コンテンツの連続投稿
まったく同じ画像・キャプションを繰り返し投稿するとスパム判定されます。
NG — 禁止コンテンツの投稿
コンテンツポリシー違反の投稿が複数回あると、警告なしで停止されることがあります。
NG — 第三者ツールの過剰使用
フォロー・いいねを自動化する非公式ツールはMetaの規約違反です（このアプリはMeta公式APIを使用しています）。
NG — コミュニティガイドライン違反の通報が集中
多くのユーザーから通報を受けると審査が入り、停止されることがあります。

警告サインと対処

こんな症状が出たら注意

  • 「アクションをブロックされました」表示 → 一時的な制限。数時間〜数日休む
  • 投稿が表示されない・リーチが極端に低い→「シャドウバン」の可能性。禁止ハッシュタグを使っていないか確認
  • ログインできなくなった→ アカウント停止。Metaのサポートに異議申し立てが可能

Threads 投稿ルール

Threads

ThreadsはMetaが運営するため、基本的にInstagramと同じコミュニティガイドラインが適用されます。

Threads 固有の注意点

  • 1投稿あたり最大500文字（このアプリでは400文字以内を推奨）
  • リンクは投稿末尾に1つ追加可能（このアプリの固定URL設定で対応）
  • ハッシュタグは現在サポートが限定的 — 多用しない方が無難
  • InstagramのBANがThreadsにも連動することがある
  • 同一内容をInstagramとThreadsに同時投稿することは問題ない

アクセス数が伸びるプロフィールの作り方

プロフィール文（Bio）

🎯

誰のための何が得られるアカウントかを明記する

「〇〇専門の△△です。□□のお役立ち情報を毎日発信」のように、ターゲット → 発信内容 → 更新頻度の3点を入れると新規フォロワーが増えやすくなります。

🔗

URLは必ず設定する

プロフィールのリンクは唯一クリック可能なURLです。サービスページ・ブログ・LINE公式アカウントへの導線として必ず設定してください。Linktree等を使って複数リンクをまとめるのも効果的です。

📌

改行・絵文字で視認性を上げる

長文より箇条書きで読みやすく。絵文字を使って各行の頭をアイコン化すると視覚的にスキャンしやすくなります。

アイコン・ハイライト・固定投稿

アイコン
顔写真（個人）または明確なロゴ（企業）を使用。背景がごちゃごちゃしていない、正方形に収まるデザインにする。
ストーリーズハイライト
「サービス紹介」「よくある質問」「お客様の声」など、新規訪問者が知りたい情報をハイライトにまとめて固定。
固定投稿（ピン）
最も見せたい投稿を最大3件ピン留め。自己紹介・人気コンテンツ・最新キャンペーン等が効果的。
ユーザーネーム
覚えやすく、ブランド名に近いものを選ぶ。記号は最小限に。他のSNSと統一すると検索されやすい。

アクセス数を伸ばす方法

ハッシュタグ戦略

効果的なハッシュタグの使い方

  • 数は5〜10個が目安（30個付けるより関連性の高いもの厳選の方が効果的）
  • 大中小を混ぜる — 大（100万件以上）：中（1〜100万）：小（1万未満）＝1：2：2 程度
  • 禁止ハッシュタグを使わない — 一部のハッシュタグはBAN対象。使う前に検索して確認
  • 業界・地域・内容に直結したタグを優先する
  • 毎回同じタグをコピペするのはスパム判定リスクあり。投稿ごとに一部変える
このアプリの「固定ハッシュタグ」設定で、全投稿に自動追加するタグをまとめて設定できます。投稿ごとに変えたいタグは記事ネタの「ハッシュタグ」欄を使ってください。

アルゴリズムに好まれる投稿

⏰

投稿時間を最適化する

フォロワーが最もアクティブな時間帯（一般的に朝7〜9時、昼12〜13時、夜19〜21時）に投稿する。インサイトで自分のフォロワーのアクティブ時間を確認して合わせる。

💬

エンゲージメントを促す

キャプション末尾に質問を入れる（「あなたはどう思いますか？」等）。コメントには必ず返信する。保存・シェアされる「有益コンテンツ」はいいねより効果的。

🎨

フィードの統一感を出す

投稿の色調・フォント・デザインを統一すると、プロフィールを訪問した際の印象が良くなりフォロー率が上がります。このアプリの「記事テイスト」設定で文体を統一できます。

📊

定期的にインサイトを確認する

どの投稿がリーチ・保存数が多いかを分析し、好まれるコンテンツを増やす。インサイトはInstagramアプリの「プロフェッショナルダッシュボード」から確認できます。

API 利用上の注意

API経由の自動投稿にも同じルールが適用されます。
Metaは投稿元がAPIかアプリかを区別しません。自動投稿でも手動投稿と同じコンテンツポリシーに従う必要があります。

API利用時の制限事項

  • Instagram API：1日あたりの投稿数は50件まで（通常の運用では問題なし）
  • Threads API：1日あたり250投稿まで
  • 画像はHTTPS経由で公開アクセス可能なURLが必要（このアプリが自動処理します）
  • 投稿に使用する画像は著作権フリーか自社所有のものを使用する
  • アクセストークンを第三者と共有しない。漏洩した場合はすぐにMeta Developerで失効させる
安全に運用するためのチェックリスト
  • ✅ Instagramがビジネスアカウントになっている
  • ✅ 投稿内容はコミュニティガイドラインに準拠している
  • ✅ ハッシュタグは禁止タグを除外した適切なものを使用
  • ✅ アクセストークンの有効期限を定期的に確認・更新している
  • ✅ 自動投稿の頻度は1日数件以内に抑えている
  • ✅ コメントへの返信など手動のエンゲージメントも行っている
